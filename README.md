# Task Manager App
This is a basic todo App. Using MERN with Login/Signup.
+ Simple Weather App

## Guide on Installation
**For Frontend**
From the client folder
```
npm install
```
**For Backend**
From the root folder
```
npm run install-all
```
**To start the App**
```
yarn dev
```
